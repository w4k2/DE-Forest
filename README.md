### Methods

- DE-Forest - the proposed method, ensemble classfier
- RandomFS - native implementation of Random Forest
- optimization - Differential Evolution optimization used in DE-Forest
- bootstrap_optimization - Differential Evolution optimization with bootstrapping used in DE-Forest
- bootstrap_optimization_with_constraints - Differential Evolution optimization with bootstrapping and with constraints used in DE-Forest

### Results

Results of experiments in form of raw results, tables of average metrics, Wilcoxon ranking.

### SMAC

smac_outputs consists the output from the SMAC3 algorithm. It contains: 

- the configuration space - hyperparameters of the method to be optimized
- the run history - successive optimization iterations along with hyperparameter values
- the scenario - parameters of the optimization algorithm and selected instances, i.e. datasets
- the statistic - final optimization statistics and selected hyperparameter values
